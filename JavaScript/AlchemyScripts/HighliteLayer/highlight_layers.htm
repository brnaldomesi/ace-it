Highlight Layers
This is probably the easiest DHMTL effect you can do. But to make it cross-browser compatible takes some knowledge. The best way to do this is to make a TABLE and use ILAYER and LAYER tags.
    <TABLE CELLPADDING=0 CELLSPACING=0 BORDER=0>
      <TR>
        <TD></TD>
      </TR>
      <TR>
        <TD></TD>
      </TR>
    <TABLE>
Just so you know, the cellpadding has to be 0 so that the highlighting in Netscape works 100%. So, we have made 2 sections for highlightable layers. The next step is to add a with to the TDs or the TABLE. But for now we'll add it to the table. So it should look like this now:
    <TABLE CELLPADDING=0 CELLSPACING=0 BORDER=0 WIDTH=150>
      <TR>
        <TD></TD>
      </TR>
      <TR>
        <TD></TD>
      </TR>
    <TABLE>
Now to get on with the highlighting. Lets add the Internet Explorer highlight. Add the onmouseover and onmouseout events to the TDs. Both should look like this now:
    <TABLE CELLPADDING=0 CELLSPACING=0 BORDER=0 WIDTH=150>
      <TR>
        <TD BGCOLOR=white ONMOUSEOVER="bgColor='yellow'" ONMOUSEOUT="bgColor='white'"></TD>
      </TR>
      <TR>
        <TD BGCOLOR=white ONMOUSEOVER="bgColor='yellow'" ONMOUSEOUT="bgColor='white'"></TD>
      </TR>
    <TABLE>
For the final step, lets add the Netscape LAYERs. Internet Explorer totally ignores them so it is safe to put them. Plus Netscape totally ignores ONMOUSEOVER and ONMOUSEOUT attributes on TD tags. Now we will have to add an ILAYER and a LAYER tag and text to each of the cells of the table. The final product should look like this:
    <TABLE CELLPADDING=0 CELLSPACING=0 BORDER=0 WIDTH=150>
      <TR>
        <TD BGCOLOR=white ONMOUSEOVER="bgColor='yellow'" ONMOUSEOUT="bgColor='white'">
          <ILAYER>
            <LAYER ONMOUSEOVER="this.bgColor='yellow'" ONMOUSEOUT="this.bgColor='white'" WIDTH="100%">
            Highlight 1
            </LAYER>
          </ILAYER>
        </TD>
      </TR>
      <TR>
        <TD BGCOLOR=white ONMOUSEOVER="bgColor='yellow'" ONMOUSEOUT="bgColor='white'">
          <ILAYER>
            <LAYER ONMOUSEOVER="this.bgColor='yellow'" ONMOUSEOUT="this.bgColor='white'" WIDTH="100%">
            Highlight 2
            </LAYER>
          </ILAYER>
        </TD>
      </TR>
    <TABLE>
